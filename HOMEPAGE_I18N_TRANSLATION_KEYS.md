## Notes

- Most of these components use API data (testimonials, portfolio items, stats), so **only section headers need translation**
- Button text that repeats (like "Read More", "Get Started") should use the same translation key throughout
- Aria-labels are accessibility features and should be translated for screen reader users
- Some text appears in multiple places - consolidate to avoid duplication (e.g., banner_heading vs hero_heading)
